


## Estructura de el proyecto
```
src/main/java/com/biblioteca/
├── config/                # Configuraciones globales (Spring Security, CORS, etc.)
├── controller/            # Controladores (Manejan las solicitudes HTTP)
├── service/               # Lógica de negocio (Servicios)
├── repository/            # Acceso a datos (Repositorios JPA)
├── model/                 # Entidades y DTOs (Modelos de datos)
├── exception/             # Manejo de excepciones personalizadas
├── dto/                   # Objetos de Transferencia de Datos (DTOs)
└── BibliotecaApplication.java # Clase principal de Spring Boot
```
